# UFS_FMS_CI

To run the UFS weather model on Orion
1.  git clone --branch orion_runscript https://github.com/mlee03/UFS_FMS_CI.git run_ufs
2.  cd run_ufs
3.  ./run_weather_model  
4.  ./run_nccmp_wm

To run the UFS S2S model on Orion
1.  git clone --branch orion_runscript https://github.com/mlee03/UFS_FMS_CI.git run_ufs
2.  cd run_ufs
3.  ./run_s2s_model
4.  ./run_nccmp_s2s
